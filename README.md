# AI-Powered Resume Agent 🚀

A serverless AI Agent built with **Google Gemini 2.5 Flash** and deployed on **Google Cloud Run**. This project was developed as part of the **Hack2Skill** Generative AI Academy challenge.

## 🌟 Features
- **Smart Summarization:** Uses Gemini 2.5 Flash to parse and summarize resume data.
- **Cloud Native:** Fully containerized using Docker and hosted on GCP Cloud Run.
- **API Integration:** Built with Python Flask and integrated with Google Generative AI SDK.

## 🛠️ Tech Stack
- **AI Model:** Gemini 2.5 Flash (2026 Edition)
- **Infrastructure:** Google Cloud Platform (GCP)
- **Framework:** Python Flask
- **Deployment:** Cloud Run (Serverless)
- **DevOps:** Docker, Git
